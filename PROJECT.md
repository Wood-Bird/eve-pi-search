# EVE行星材料查询工具

## 项目概述

基于EVE Echoes行星材料统计数据，开发的手机友好型查询工具，帮助玩家快速查找最佳材料采集位置。

## 已实现功能

### 1. 三种搜索模式

| 模式 | 说明 |
|------|------|
| 按星座查询 | 选择星域→星座，查看该星座所有材料产出 |
| 按星系搜索 | 输入星系名称（如Y-2ANO），自动定位到对应星座 |
| 按材料搜索 | 输入材料名称（如聚芳酰胺），查找最佳产出星座 |

### 2. 模糊搜索下拉

- 输入时实时显示匹配结果
- 显示位置信息：星域 › 星座 › 星系
- 点击自动定位并查询

### 3. 工业中心管理

- 支持添加多个工业中心
- 星系搜索定位模式：输入星系自动定位星域和星座
- 手动选择模式：传统下拉框选择
- 设置保存在本地存储，下次访问自动加载

### 4. 智能排序

查询结果按以下优先级排序：
1. 同行星（最高优先级）
2. 同星座（可一键回收）
3. 同星域
4. 产量高的优先

### 5. 动态星空背景

- 150颗随机分布的星星
- 星云层缓慢漂移效果
- 青色发光星星（呼应EVE主题）
- 周期性流星效果

## 数据结构

### index.json 索引文件

```json
{
  "regions": ["源泉之域", "欧米斯特", ...],          // 56个星域
  "constellations": ["巴西尔斯克", "RL36-K", ...],   // 649个星座
  "materials": ["有毒金属", "稀有气体", ...],         // 34种材料
  "systems": ["Y-2ANO", "VI2K-J", ...],              // 4,512个星系
  "planet_constellation_map": {                      // 36,827个行星映射
    "Y-2ANO 十": { "region": "源泉之域", "constellation": "巴西尔斯克" }
  },
  "system_constellation_map": {                      // 星系→星座映射
    "Y-2ANO": { "region": "源泉之域", "constellation": "巴西尔斯克" }
  },
  "system_planets": {                                // 星系→行星列表
    "Y-2ANO": ["Y-2ANO 十", "Y-2ANO 四", ...]
  }
}
```

### constellation_*.json 星座数据文件

每个星座一个文件，约14KB：

```json
{
  "region": "源泉之域",
  "constellation": "巴西尔斯克",
  "materials": {
    "有毒金属": {
      "max_output": 13.59,
      "count": 5,
      "records": [
        { "planet": "Y-2ANO 十", "richness": "完美", "output": 13.59 },
        ...
      ]
    }
  }
}
```

## 行星名称格式

行星名称格式为：`星系名称 + 空格 + 星球编号`

示例：
- `Y-2ANO 十` → 星系：Y-2ANO，星球编号：十
- `VI2K-J 六` → 星系：VI2K-J，星球编号：六

## 使用方法

### 启动本地服务器

```bash
cd E:\claude-code-demo
python -m http.server 8080
```

### 访问地址

```
http://localhost:8080/query.html
```

**注意**：必须通过HTTP服务器访问，直接打开文件会因为CORS问题无法加载JSON数据。

### 手机访问

同一局域网下，手机浏览器访问：
```
http://[电脑IP]:8080/query.html
```

## 开发历程

### 需求确认

1. 用户需求：手机上查询行星材料
2. 核心逻辑：同星座优先（可一键回收）> 同星域 > 产量
3. 多工业中心：用户可能有多个工业中心
4. 星系搜索：用户对星系名称熟悉，需要模糊搜索定位

### 技术挑战

1. **数据量问题**：原始Excel 147,936条记录，JSON文件13MB太大
   - 解决：按星座拆分为649个文件，每个约14KB

2. **CORS问题**：本地打开HTML无法加载JSON
   - 解决：必须通过HTTP服务器访问

3. **星系识别**：行星名称包含星系信息
   - 解决：提取空格前的部分作为星系名称，建立逆向映射
